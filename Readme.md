WeatherApp demo made in Kotlin Programming language.

Features Used are:
MVVM,

RxJava,

Realm Database for offline handling,

ViewBinding

ViewModel

OpenWeatherMap api for weather data.

Coil  for image loading https://github.com/coil-kt/coil
